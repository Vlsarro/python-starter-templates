# Simple Flask app (Python 3.7)

## Linux:
1. Создание и активация виртуального окружения: 
    * cd путь к корню проекта
    * python -m venv venv
    * source venv/bin/activate

2. Установка зависимостей:
    * ```pip install -r requirements.txt```

3. Запуск веб-сервиса:
    * python app.py
    
4. Запуск локально:
    * https://0.0.0.0:5000
